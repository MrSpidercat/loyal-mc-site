Announcements:
We're switching to the MultiMC launcher for our modpacks! [The option to download the modpack files for installation on the regular Minecraft Launcher is coming soon.]
Our reasoning behind switching to MutliMC
  • MultiMC is lightweight and runs on multiple operating systems.
  • MutlIMC is very easy to use after you get used to it.
  • MultiMC is open source.
  • MutliMC has Minecraft world, mod, and resource pack management.
  • It removes the requirement for people to install Java and Fabric beforehand
  • It allows us to setup a recommended configuration for first time users.

How to install the Loyal MC Modpack
  1. Download and install MultiMC.
  2. Launch MutliMC.
  3. To login press the settings button located at the top, select the accounts tab, on the right select Add Mojang or Add Microsoft, enter your credentials and login.
  4. Select your account in the top right on the main page.
  5. Select Add Instance in the top left.
  6. Name the Instance.
  7. Select Import from zip.
  8. Download the Modpack under '1.17.1 MultiMC Instance Download' then select browse then select the .zip file you downloaded.
  9. Select OK.
  10. Select the Instance then select Launch.
You have successfully installed the Loyal MC Modpack!
