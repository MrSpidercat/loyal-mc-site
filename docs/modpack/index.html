Announcements:
We're switching to the PolyMC launcher for our modpacks! [The option to download the modpack files for installation on the regular Minecraft Launcher is coming soon.]
Our reasoning behind switching to PolyMC
  • PolyMC is lightweight and runs on multiple operating systems.
  • PolyMC is very easy to use after you get used to it.
  • PolyMC is open source.
  • PolyMC has Minecraft world, mod, and resource pack management.
  • It removes the requirement for people to install Java and Fabric beforehand
  • It allows us to setup a recommended configuration for first time users.

How to install the Loyal MC Modpack
  1. Download and install PolyMC.
  2. Launch PolyMC.
  3. To login press the settings button located at the top, select the accounts tab, on the right select Add Mojang or Add Microsoft, enter your credentials and login.
  4. Select your account in the top right on the main page.
  5. Select Add Instance in the top left.
  6. Name the Instance.
  7. Select Import from zip.
  8. Download the Modpack under '1.18.2 PolyMC Instance Download' then select browse then select the .zip file you downloaded.
  9. Select OK.
  10. Select the Instance then select Launch.
You have successfully installed the Loyal MC Modpack!
